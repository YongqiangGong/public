欢迎回来，我的朋友们。今天我们继续更新TCGA数据库的相关内容，或许下午会更新一些关于基础科研系列的文章，大家拭目以待。

话不多说，来到正题。首先先加载我写好的函数，用于下载临床数据的。

tcga_clinical <- function(x){
  y <- str_c("TCGA-",x)
  clin.coad <- TCGAbiolinks::GDCquery_clinic(y, "clinical")
  clinical <- clin.coad
  message("临床数据下载成功！(nkyqgong)")
  return(clinical)
}

接下来我们开始实战操作，例子是KIRP项目，大家自己选择项目就可以。

library(tidyverse)
project <- "KIRP"
clinical <- tcga_clinical(project)

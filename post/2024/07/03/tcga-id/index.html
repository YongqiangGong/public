欢迎回来，我的朋友们。今天我们继续更新TCGA数据库的相关内容。

话不多说，来到正题。首先先加载我写好的函数，用于转换基因ID。

tcga_cbind <- function(counts){
  mrna <- counts$mrna
  lnc <- counts$lnc
  mrna <- cbind(mrna,counts$symbol_mrna) %>% 
    select(-ensembl) %>% 
    select(gene,everything())
  lnc <- cbind(lnc,counts$symbol_lnc) %>% 
    select(-ensembl) %>% 
    select(gene,everything())
  result <- list(mrna=mrna,lnc=lnc)
  return(result)
}

接下来我们就要用到之前保存的数据了，下面我把前几期用到的函数整理一下，直接运行下载就可以。

counts <- tcga_download(project)
counts <- tcga_cbind(counts)
clinical <- tcga_clinical(project)
snp <- tcga_snp(project)
save(counts,snp,clinical,
     file = "Data.Rda")

运行之后我们就可以得到整理好的原始数据了并且保存下来，下一期会给大家整理怎么过滤矩阵，提取重要的临床数据。

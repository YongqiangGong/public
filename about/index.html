龚永强在无聊的时候开发了一些Shiny Apps，或许对你会有一些帮助。

List

-Venn Diagram

-Gene-ID Transform

-Pubmed Hotspot

-GO/KEGG Enrichment Analysis

Description

-Venn Diagram：你可以使用它绘制韦恩图并且下载，最多可以上传4个数据集进行绘制，可以更改标题和百分比大小。下载时速度比较慢，因为设置的清晰度比较高。（注意输入形式用逗号或者空格隔开）

-Gene-ID Transform：你可以使用它转换基因ID，主要包括四种形式（Symbol，Ensembl，Entrezid和Gene Name），转换成功后可以下载列表。（注意输入形式用逗号或者空格隔开）

-Pubmed Hotspot：你可以用它查找Pubmed相关关键词出现的次数，对你的研究或许会有一些启发。输入你的关键词和想要查找的期刊，跳出相关关键词，你可以设置最大文章量以及要查找的时间范围。

-GO/KEGG Enrichment Analysis：你可以利用它进行GO/KEGG分析，并且提供了简单的柱形图和气泡图，结果可以下载下来进行下一步分析。（注意输入形式用逗号或者空格隔开）

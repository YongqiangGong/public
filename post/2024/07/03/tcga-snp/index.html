TCGA：如何下载SNP数据？

Jenking Gong

2024/07/03

欢迎回来，我的朋友们。今天我们继续更新TCGA数据库的相关内容。

首先先加载我写好的函数，用于下载SNP数据的。

tcga_snp <- function(x){
  y <- str_c("TCGA-",x)
  query_snp <- TCGAbiolinks::GDCquery(
    project = y,
    data.category = "Simple Nucleotide Variation",
    data.type = "Masked Somatic Mutation",
    access = "open"
  )
  TCGAbiolinks::GDCdownload(query_snp)
  snp <- TCGAbiolinks::GDCprepare(query_snp)
  message("体细胞突变数据下载成功！(nkyqgong)")
  return(snp)
}

接下来我们开始实战操作，例子是KIRP项目，大家自己选择项目就可以。

library(tidyverse)
project <- "KIRP"
snp <- tcga_snp(project)

接下来我们安心等待结果就可以了。本期结束！
